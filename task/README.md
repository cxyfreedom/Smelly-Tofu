## 实践方法

 - 每个人在每个`task`下建立一个自己账户名称的`workspace`,比如
```
task01
 ├─cxyfreedom
```

 - 在该目录下添加三部分内容
    - 自己的代码，统一放在`code`目录下
    - 学习笔记，统一用`note.md`
    - 相互review，统一用`review.md`（其他队员把对该成员的代码进行review，把优缺点都写一下）

 - 除了代码以外的文档默认使用`markdown`语法编写
 - 最后把各个成员的代码合并成一份，我会放在每个`task`下的`final`目录
